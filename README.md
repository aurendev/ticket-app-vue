### Ticket-App

#### Guia de instalacion

1. Ejecutar `composer install`

2. Configurar base de datos y ejecutar `php artisan migrate --seed`

3. Ejecutar `npm run dev`

##### Diagrama entidad-relacion y db.sql
estos archivos se encuentran en la raiz del proyecto :

- `./digrama.png` Diagrama entidad-relacion.
- `./database.sql ` base de datos.
